# 💳 체크인/결제 시스템 완전 테스트 케이스 (80개)

## 📋 테스트 영역 개요

체크인/결제 시스템은 실제 금전 거래와 관련된 크리티컬한 영역으로, 100% 정확성과 무결성이 보장되어야 합니다.

### 테스트 범위
1. 체크인 프로세스 (15개)
2. 결제 상태 관리 (15개)
3. 시간/금액 조정 (20개)
4. 노쇼 처리 (10개)
5. 환불 프로세스 (10개)
6. 체크인 취소 (10개)
