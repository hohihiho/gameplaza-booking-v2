# 광주 게임플라자 예약 시스템 완전 기획서 (최종)

## 📋 프로젝트 개요

### 목적
- 카카오톡 오픈채팅 기반 예약을 **웹 기반 PWA**로 전환
- **모바일 퍼스트** 설계로 99% 모바일 사용자 최적화
- **실시간 알림**과 **통합 관리**로 운영 효율성 극대화
- **1인 1대 대여** 원칙으로 간소화된 예약 시스템

### 핵심 특징
- 🔐 **구글 OAuth + 전화번호 인증** (Firebase FCM)
- 📱 **PWA 웹앱** (홈화면 추가, 실시간 알림)
- 🔧 **노코드 빌더** 스타일 관리자 시스템
- ⚡ **실시간 동기화** (Supabase Realtime)
- 🎮 **리듬게임 특화** 기능 (기종별 2P, 점검시간 등)

---

## 🏗️ 기술 스택

### Frontend
- **React** (PWA 지원)
- **Tailwind CSS** (모바일 우선 반응형)
- **Supabase Client** (실시간 DB 연동)

### Backend & Database
- **Supabase** (PostgreSQL + Realtime + Storage)
- **Firebase FCM** (푸시 알림)
- **구글 OAuth** (인증)

### 배포 & 호스팅
- **Vercel** (Frontend 호스팅)
- **커스텀 도메인** 연결

---

## 👥 사용자 인증 & 권한

### 인증 프로세스
1. **구글 OAuth 로그인** (ONLY 구글)
2. **Firebase FCM 전화번호 인증** (SMS)
3. **인증 완료** → 모든 기능 이용 가능

### 사용자 권한
| 권한 | 접근 가능 기능 |
|------|----------------|
| **일반 사용자** | 예약 신청, 마이페이지, 정보 조회 |
| **스태프** | 예약 조회, 기기현황 확인 (읽기 전용) |
| **관리자** | 모든 기능 + 체크인 + 계좌정보 관리 |

## 🎯 핵심 기능 요약

### 1. 📝 예약 신청 시스템
- 24시간 룰 적용된 예약 신청
- 실시간 예약 가능 여부 체크
- 1인 1대 원칙 (기기 번호 선택)
- 마이마이 2P 옵션 (+10,000원)
- 관리자 수동 승인/거절

### 2. ✅ 체크인 & 결제 시스템
- 기기 번호 자동 배정
- **혁신적 계좌이체 시스템**: FCM 알림으로 계좌정보 전송
- 백업 시스템: 알림 문제시 관리자 폰 화면 직접 보여주기
- 현금/계좌이체 결제 지원

### 3. 🎮 기기 관리 시스템 (2개 시스템 분리)
- **기기 관리**: 전체 오락실 기기 현황 관리
- **대여 관리**: 예약 가능 기기만 별도 관리
- 드래그앤드롭 순서 변경
- 대여 종류별 가격 설정

### 4. 📝 콘텐츠 관리 시스템 (노코드 빌더)
- 홈페이지, 오락실 이용안내, 예약 이용안내, 대여안내 페이지 편집
- 드래그앤드롭 섹션 관리
- 실시간 편집 및 즉시 반영
- 모바일 미리보기
- 영업시간, 요금표, FAQ 등 동적 콘텐츠 관리

### 5. 📊 관리자 통계 & 데이터 관리
- 실시간 현황 대시보드
- 기간별/기기별/고객별 분석
- 엑셀/CSV/구글시트 연동
- 정산 시스템 호환

### 6. 👥 고객 검색 & 관리
- 이름/전화번호/이메일/닉네임 검색
- 24시간 이내 수동 예약/취소 (관리자 권한)
- 이메일 기반 블랙리스트 관리

### 7. 📱 알림 시스템 (Firebase FCM)
- 예약 승인/거절 즉시 알림
- 계좌이체 요청 실시간 알림
- 대여 1시간 전 리마인더
- 관리자 신규 예약 알림
