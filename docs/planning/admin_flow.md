## 📋 예약 관리 플로우

```
예약 관리 시스템 접속
    ↓
신규 예약 신청 목록 확인
    ↓
예약 상세 정보 확인
- 고객 정보 (이름, 연락처)
- 예약 일시 및 시간대
- 선택 기기 및 대수
- 특별 요청사항
    ↓
승인/거절 결정
    ↓
┌─────────────────┬─────────────────┐
│      승인       │      거절       │
│       ↓         │       ↓         │
│ 기기 번호 자동   │ 거절 사유 입력   │
│ 배정 처리       │       ↓         │
│ (순서대로 배정)  │ 거절 처리 및    │
│       ↓         │ FCM 알림 발송   │
│ 승인 처리 및    │       ↓         │
│ FCM 알림 발송   │ 예약신청중     │
│       ↓         │ 상태 해제      │
│ 기기 가예약     │               │
│ 상태 변경       │               │
└─────────────────┴─────────────────┘
```

## ✅ 체크인 시스템 플로우

```
체크인 시스템 접속
    ↓
승인된 예약 목록 확인
    ↓
고객 도착 시 신원 확인
    ↓
배정된 기기 번호 안내
    ↓
결제 방식 선택
    ↓
┌─────────────────┬─────────────────┐
│      현금       │    계좌이체      │
│       ↓         │       ↓         │
│ 현금 수령 체크   │ 관리자 계좌번호  │
│       ↓         │ 표시 및 고객에게 │
│ 체크인 완료     │ FCM 알림 발송   │
│     처리       │       ↓         │
│               │ 백업 시스템:     │
│               │ 알림 문제시     │
│               │ 관리자 폰 화면   │
│               │ 직접 보여주기   │
│               │       ↓         │
│               │ 관리자 추가 옵션: │
│               │ - 계좌정보 재발송 │
│               │ - 결제 확인     │
│               │       ↓         │
│               │ 은행앱에서      │
│               │ 입금 확인 후    │
│               │ 수동 완료 처리   │
│               │       ↓         │
│               │ 체크인 완료     │
│               │     처리       │
└─────────────────┴─────────────────┘
    ↓
기기 상태 '대여중'으로 변경
    ↓
대여 시작/종료 시간 기록
    ↓
실시간 현황 업데이트
```

## 🎮 기기 관리 시스템

```
기기 관리 시스템 접속
    ↓
전체 기기 목록 관리
    ↓
기기 관리 작업 선택
    ↓
┌─────────┬─────────┬─────────┬─────────┐
│기기 추가 │기기 편집 │순서 변경 │상태 관리 │
│   ↓     │   ↓     │   ↓     │   ↓     │
│새 기기  │기기정보 │드래그앤  │개별기기 │
│등록     │수정     │드롭순서 │상태변경 │
│기종/모드│       │조정     │        │
│요금/대수│       │        │        │
│   ↓     │   ↓     │   ↓     │   ↓     │
│카테고리 │변경사항 │변경사항 │기기상태 │
│자동분류 │저장    │저장    │선택    │
└─────────┴─────────┴─────────┴─────────┘
    ↓
기기 상태 선택 (상태 관리시)
    ↓
┌─────────┬─────────┬─────────┬─────────┐
│  정상   │  점검   │  고장   │ 대여중  │
│   ↓     │   ↓     │   ↓     │   ↓     │
│사용가능 │점검중   │고장    │현재대여 │
│상태    │(예약차단)│(예약차단)│정보표시 │
└─────────┴─────────┴─────────┴─────────┘
    ↓
예약 가능 여부 설정
    ↓
예약 가능 기기 목록 업데이트
```

## 💰 대여 관리 시스템

```
대여 관리 시스템 접속
    ↓
예약 가능 기기 목록 로드
(기기관리에서 연동)
    ↓
대여 설정 작업 선택
    ↓
┌─────────┬─────────┬─────────┬─────────┐
│시간대설정│대여종류 │가격설정 │2P설정  │
│   ↓     │   ↓     │   ↓     │   ↓     │
│기종별   │고정크레딧│종류별   │2P가능  │
│대여가능 │프리플레이│가격책정 │여부및  │
│시간대   │무한크레딧│        │추가요금 │
│   ↓     │   ↓     │   ↓     │   ↓     │
│점검시간 │기종별   │시간대별/│마이마이 │
│고려한   │사용가능한│종류별  │2P:     │
│시간대   │대여종류 │개별가격 │+10,000원│
│입력    │선택    │        │등      │
└─────────┴─────────┴─────────┴─────────┘
    ↓
설정 저장 및 예약시스템 반영
```
