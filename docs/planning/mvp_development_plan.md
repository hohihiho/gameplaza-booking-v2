## 📅 MVP 개발 일정 (3주)

### 1주차: 기본 구조
- Supabase 프로젝트 설정
- 구글 OAuth 연동
- 기본 페이지 구조 (홈/예약/관리자)
- 반응형 레이아웃 기본 구조

### 2주차: 핵심 기능
- 예약 신청/조회 기능
- 관리자 예약 승인/거절
- 기기 현황 표시 (고정 데이터)
- 체크인 기능

### 3주차: 마무리
- 사용자 테스트 및 버그 수정
- 기본적인 에러 처리
- Vercel 배포
- 기본 사용법 안내
